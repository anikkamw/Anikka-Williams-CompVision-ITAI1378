# Model Documentation

This folder contains the weights and architecture details for the MangeVision detection system.

- **Primary Model:** `best.pt` (YOLOv8 Nano)
- **Framework:** Ultralytics YOLOv8
- **Input Size:** 640x640
- **Training Epochs:** 25
- **Performance:** 2.8ms inference speed. 
- **Training Data:** Custom-tuned on the Dog Skin Diseases and Human Scabies datasets to optimize for Demodicosis detection.

*Note: Large weight files are managed via the notebook or external links to prevent repository bloat.*
